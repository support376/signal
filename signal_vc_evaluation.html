1. Executive Verdict

7.4 / 10
Conditional PASS — Lead 가능 (조건부)

Pre-seed에 작게 들어가고, Phase 2 검증 마일스톤 통과 시 Seed lead 옵션 확보.

Signal은 진짜로 흥미로운 카테고리 베팅이다. "잠재의식 측정 + 호환성 + 학술 깊이 + 반(反)광고 mission"의 4중 결합은 우리가 본 deck 중에서 가장 신선한 카테고리 정의 중 하나다. 그러나 매출 모델, retention 가설, founder dependency, 글로벌 확장 가능성에 대해 검증되지 않은 가정이 너무 많다. Pre-seed로 진입하고, Phase 2 정량 검증을 watch-list로 잡은 후 Seed lead로 follow-on 하는 구조가 risk-adjusted optimal이다.

2. Strengths — 왜 들어갈 만한가

2.1 카테고리 정의의 우아함 9/10

"4개 시장의 교집합이 비어 있다"는 frame은 정확하다. 데이팅앱·MBTI·소셜·Creator economy 모두 한 axis씩 부족했고, Signal은 그 axis들을 "잠재의식 + 호환성 + 정서"로 묶어낸다. 이 카테고리 framing 자체가 venture-scale 가능성의 신호다.

2.2 제품의 위장 불가능성 8/10

LLM이 측정 대상을 사용자에게 숨기는 구조는 진짜 차별점이다. MBTI가 self-report라는 본질적 결함을 가진 반면, Signal의 3 Layer 구조는 "의식적 조작 불가능 + 결정적 수학 + 문학적 깊이"를 동시에 충족한다. 검증된 페르소나 결과(이인혁: "타인을 믿는 것보다 시스템을 배신하는 것을 더 두려워하는")는 GPT 시대 이전엔 만들 수 없었던 quality다.

2.3 학술적 엄밀함 8/10

38축 → 15축으로 축소한 과정이 인상적이다. 임의로 만든 축이 없고, 모두 cross-cultural 검증된 표준(Schwartz, Big Five, Bowlby, Haidt, Gottman)에 anchor되어 있다. 향후 학계·B2B(HR, 상담) 진입 시 defensible authority를 만든다.

2.4 Anti-광고 Mission의 strategic value 9/10

이건 단순한 marketing이 아니라 moat다. Big tech(Meta, Google)이 같은 카테고리에 진입해도 "우리는 광고에 정서 데이터를 쓰지 않는다"는 약속이 신뢰받기 어렵다. Signal은 day 1부터 그 약속의 신뢰성을 brand에 baked-in 한다. Sequoia가 Notion·Figma·Linear에서 본 것과 같은 종류의 "founder-belief moat".

2.5 한국 GTM의 구조적 우위 8/10

KakaoTalk 100% 점유 + MBTI/사주 문화 + K-pop 글로벌 entry vehicle은 한국이 진짜 "unfair 글로벌 entry market"임을 만든다. Co-Star가 미국에서 30M DL을 만들었던 것을 감안하면, Signal이 한국에서 1M PMF → 일본/미국 K-pop 팬덤으로 jump 하는 trajectory는 plausible.

2.6 Founder의 framing 능력 8/10

사업 본질을 "두 길 중 하나를 선택했다"로 정의하고, day 1 anti-pattern 5가지를 영구히 거절하는 규율은 mission-driven founder의 시그니처다. Patrick Collison, Alexandr Wang, Daniel Ek에서 본 것과 같은 framing discipline. 이 quality는 자금 배가에 가장 잘 반응한다.

3. Concerns — 왜 망설이는가

3.1 매출이 어디서 진짜 나오는가의 미검증 RED FLAG

  • Pricing tier 구조는 합리적으로 보이지만, "왜 사용자가 $7/월을 자발적으로 낼 것인가"의 가설이 약하다.
  • Self-discovery 앱의 함정: 결과가 영구적이라 재방문 동기가 약하다. Co-Star/16Personalities의 LTV는 일반적으로 $5–12. $30 LTV는 매우 낙관적.
  • Premium → Premium Plus 5x conversion 가정도 freemium 표준 대비 낙관적.
검증 요청: Phase 2 종료 시 free → casual conversion 실제 측정. >15% 미만이면 모델 재설계 필요.

3.2 Creator Cold Start의 비대칭 위험 RED FLAG

  • 사업 전체가 "첫 mid-tier creator를 잡는다"는 단일 의존 가설에 걸려 있다.
  • Cameo가 launch 시 같은 문제를 겪었고, 결국 $1M 보장 deal × 5명 정도의 자금이 필요했다. Signal의 $10–15k는 mid-tier만 가능하고, mid-tier viral은 mega-tier 대비 한 자릿수 낮다.
  • 첫 creator가 reputation risk를 감수하고 "내 잠재의식 분석을 fan에게 판다"는 결정을 할 가능성은 회의적.
검증 요청: Pre-seed 자금 사용 전, 5명 mid-tier 컨택 → 응답률 측정. <20%면 GTM 재설계.

3.3 Retention의 본질적 약점 YELLOW FLAG

  • Self profile 1회 + 친구 케미 평균 3–5회 = total session 10회 미만 lifetime이 base case.
  • Co-Star의 daily horoscope는 매일 새로운 input이 있는 반면, Signal은 "새 사람을 만났을 때"에만 발생.
  • DAU/MAU < 10%일 가능성 높음.

3.4 LLM 비용 곡선 YELLOW FLAG

  • 5분 대화 × 5개 시나리오 × 100M 사용자 = 약 25 billion LLM calls. 약 $250M+/년 비용. ARR $320M 모델에서 매출의 75%가 LLM 비용.
  • Year 3 전에는 gross margin 30–40% 가능성. SaaS valuation multiple 직접 압박.
검증 요청: 사용자당 LLM 비용 실측. >$0.50 / lifetime이면 모델 재설계.

3.5 글로벌 확장의 언어/문화 리스크 YELLOW FLAG

15축은 cross-cultural 검증되었지만, narrative quality는 언어별로 완전히 새로 검증해야 한다. 융·라캉 수준의 한국어 narrative가 다른 언어로 옮겨질 때 quality가 유지될지 불확실. Co-Star도 미국 외 시장 진입 실패.

3.6 Founder Dependency YELLOW FLAG

현재 founder 1인 + 외부 개발 구조. CTO 부재. Phase 2 검증을 1인이 6개월 안에 해낼 수 있는지가 critical path.

3.7 윤리·규제 리스크 YELLOW FLAG (낮지만 catastrophic tail)

잠재의식 데이터의 유출 / 미성년자 사용 / 자살 신호 오감지 등은 brand에 영구 손상을 줄 수 있는 tail risk. GDPR/CCPA + 한국 개인정보보호법 day 1 substantial compliance 필요. Pre-seed 예산에서 법무 항목이 너무 작게 잡혀 있음.

4. Diligence Questions — Founder에게 직접 묻는 것

  1. Retention 가설: Self profile + 친구 5명 케미 후 90일 retain hook이 무엇인가? "daily reason to return"이 무엇인가?
  2. 첫 Creator: 이미 컨택 시작했는가? 응답률은? sample이 있는가?
  3. Pair 검증 결과: 5–10쌍 실측 시 false positive/negative < 20% baseline이 있는가?
  4. LLM Unit Economics: 사용자 1명당 lifetime LLM 비용 실측이 얼마인가? Cache hit ratio target은?
  5. CTO 영입: 누구를 candidate로 보고 있는가?
  6. Big Tech 진입 시점: Meta가 같은 카테고리에 진입한다면 response가 무엇인가?
  7. B2B 매출 (Year 4–5): $100M B2B ARR의 buyer는 누구인가?
  8. Exit 시나리오: IPO vs Acquisition. Strategic acquirer로 누구를 상정하는가?

Final Recommendation

1단계: Pre-seed $100k 진입 at $4–6M post.

2단계: Phase 2 종료 시 5개 정량 게이트 통과 여부 확인:

  1. Pair 검증: 5쌍 중 3쌍 이상 "우리 관계 맞다" 4+/5
  2. Creator 응답률: 컨택 20명 중 4명 이상 응답
  3. False positive/negative: 둘 다 < 25%
  4. 사용자당 LLM cost projection: < $0.50 / lifetime
  5. CTO candidate: shortlist 3명 이상 + 1명 verbal commit

3단계: 5개 중 4개 통과 → Seed $1.5M lead at $10–12M post (15% ownership)
