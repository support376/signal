## 1. Executive Verdict

**Score: 7.4 / 10**
**Decision: Pre-seed에 작게 들어가고, Phase 2 검증 마일스톤 통과 시 Seed lead 옵션 확보**

Signal은 *진짜로* 흥미로운 카테고리 베팅이다. *"잠재의식 측정 + 호환성 + 학술 깊이 + 반(反)광고 mission"* 의 4중 결합은 우리가 본 deck 중에서 가장 신선한 카테고리 정의 중 하나다. 그러나 매출 모델, retention 가설, founder dependency, 글로벌 확장 가능성에 대해 **검증되지 않은 가정**이 너무 많다. **Pre-seed로 진입**하고, **Phase 2 정량 검증**을 watch-list로 잡은 후 Seed lead로 follow-on 하는 구조가 risk-adjusted optimal이다.

## 2. Strengths (왜 들어갈 만한가)

### 2.1 카테고리 정의의 우아함 — 9/10
*"4개 시장의 교집합이 비어 있다"* 는 frame은 정확하다. 데이팅앱·MBTI·소셜·Creator economy 모두 한 axis씩 부족했고, Signal은 그 axis들을 *"잠재의식 + 호환성 + 정서"* 로 묶어낸다. **이 카테고리 framing 자체가 venture-scale 가능성의 신호**다.

### 2.2 제품의 위장 불가능성 — 8/10
LLM이 측정 대상을 사용자에게 숨기는 구조는 진짜 차별점이다. MBTI가 self-report라는 본질적 결함을 가진 반면, Signal의 3 Layer 구조는 *"의식적 조작 불가능 + 결정적 수학 + 문학적 깊이"* 를 동시에 충족한다. 검증된 페르소나 결과 (이인혁: *"타인을 믿는 것보다 시스템을 배신하는 것을 더 두려워하는"*)는 GPT 시대 이전엔 만들 수 없었던 quality다.

### 2.3 학술적 엄밀함 — 8/10
38축 → 15축으로 축소한 과정이 인상적이다. 임의로 만든 축이 없고, 모두 cross-cultural 검증된 표준 (Schwartz, Big Five, Bowlby, Haidt, Gottman)에 anchor되어 있다. 이는 향후 학계·B2B (HR, 상담) 진입 시 **defensible authority**를 만든다.

### 2.4 Anti-광고 Mission의 strategic value — 9/10
이건 단순한 marketing이 아니라 **moat**다. Big tech (Meta, Google)이 같은 카테고리에 진입해도 *"우리는 광고에 정서 데이터를 쓰지 않는다"* 는 약속이 신뢰받기 어렵다. Signal은 day 1부터 그 약속의 신뢰성을 brand에 baked-in 한다. Sequoia가 Notion·Figma·Linear에서 본 것과 같은 종류의 *"founder-belief moat"*.

### 2.5 한국 GTM의 구조적 우위 — 8/10
KakaoTalk 100% 점유 + MBTI/사주 문화 + K-pop 글로벌 entry vehicle은 한국이 진짜 *"unfair 글로벌 entry market"* 임을 만든다. Co-Star가 미국에서 30M DL을 만들었던 것을 감안하면, Signal이 한국에서 1M PMF → 일본/미국 K-pop 팬덤으로 jump 하는 trajectory는 plausible.

### 2.6 Founder의 framing 능력 — 8/10
사업 본질을 *"두 길 중 하나를 선택했다"* 로 정의하고, day 1 anti-pattern 5가지를 영구히 거절하는 규율은 **mission-driven founder의 시그니처**다. 우리가 Patrick Collison, Alexandr Wang, Daniel Ek에서 본 것과 같은 framing discipline. 이 quality는 자금 배가에 가장 잘 반응한다.

## 3. Concerns (왜 망설이는가)

### 3.1 *"매출이 어디서 진짜 나오는가"* 의 미검증 — RED FLAG
- Pricing tier 구조는 합리적으로 보이지만, *"왜 사용자가 $7/월을 자발적으로 낼 것인가"* 의 가설이 약하다.
- Self-discovery 앱의 함정: **결과가 영구적이라 재방문 동기가 약하다**. Sponsor는 시나리오 추가/Creator subscription을 해결책으로 제시하지만, Co-Star/16Personalities의 LTV 데이터를 보면 self-discovery 앱의 LTV는 일반적으로 $5–12에 머무른다. **$30 LTV는 매우 낙관적 가정**.
- Premium → Premium Plus 5x conversion 가정도 freemium 표준 (보통 10–20% Premium → 1–3% top tier) 대비 낙관적.

> **검증 요청**: Phase 2 종료 시 free → casual conversion 실제 측정. **>15% 미만이면 모델 재설계 필요**.

### 3.2 Creator Cold Start의 비대칭 위험 — RED FLAG
- 사업 전체가 *"첫 mid-tier creator를 잡는다"* 는 단일 의존 가설에 걸려 있다. $10–15k 보장 deal로 5명을 잡는다는 plan은 좋지만, **첫 1명이 결과를 인스타·유튜브에 자랑할 인센티브가 충분한가**가 핵심.
- Cameo가 launch 시 같은 문제를 겪었고, 결국 *"$1M 보장 deal × 5명"* 정도의 자금이 필요했다. Signal의 $10–15k는 mid-tier만 가능하고, mid-tier가 일으키는 viral은 mega-tier 대비 한 자릿수 낮다.
- *"Creator 노력이 0이고 평생 패시브"* 라는 hook은 강력하지만, **첫 creator가 reputation risk를 감수하고 *"내 잠재의식 분석을 fan에게 판다"* 는 결정을 할 가능성**은 회의적.

> **검증 요청**: Pre-seed 자금 사용 전, 5명 mid-tier 컨택 → 응답률 측정. **<20%면 GTM 재설계**.

### 3.3 Retention의 본질적 약점 — YELLOW FLAG
- Self profile 1회 + 친구 케미 평균 3–5회 = total session 10회 미만 lifetime이 base case.
- *"오늘 누구와 비교해볼까"* 의 daily habit이 정말 형성될 것인가? Co-Star의 daily horoscope는 매일 새로운 *"input"* 이 있는 반면, Signal의 input은 *"새 사람을 만났을 때"* 에만 발생.
- Group chemistry / Creator chemistry / 시나리오 추가가 해결책이지만, 이것들은 모두 *"새 콘텐츠가 추가될 때만"* trigger가 발생. **DAU/MAU < 10%일 가능성 높음**.

### 3.4 LLM 비용 곡선 — YELLOW FLAG
- 5분 대화 × 5개 시나리오 × 100M 사용자 = 약 25 billion LLM calls. Claude Sonnet 4.5 기준으로 $250M+/년 비용. ARR $320M 모델에서 매출의 75%가 LLM 비용.
- Cache + 자체 모델 fine-tuning 가능성은 인정하지만, **Year 3 전에는 gross margin이 30–40%에 머무를 가능성** 높음. 이는 SaaS valuation multiple을 직접 압박.

> **검증 요청**: Phase 2 종료 시 사용자당 LLM 비용 실측. **>$0.50 / lifetime이면 모델 재설계 필요**.

### 3.5 글로벌 확장의 언어/문화 리스크 — YELLOW FLAG
- 15축 자체는 cross-cultural 검증되었지만, **narrative quality는 언어별로 완전히 새로 검증해야 한다**. 융·라캉 수준의 한국어 narrative가 영어/일본어/스페인어로 옮겨질 때 quality가 유지될지 불확실.
- 한 국가의 PMF가 다른 국가로 전이되지 않을 가능성이 실재. (Co-Star도 미국 외 시장 진입 실패)

### 3.6 Founder Dependency — YELLOW FLAG
- 현재 founder 1인 + 외부 개발 구조. CTO 부재. Phase 2 검증 (시나리오 5개 양산 + pair 검증)을 1인이 6개월 안에 해낼 수 있는지가 critical path.
- Mission-driven brand가 채용 자석이라는 가설은 동의하지만, top talent가 *"founder 1인에게 baked-in 된 vision"* 에 합류할 risk를 감수할지는 별개 문제.

### 3.7 윤리·규제 리스크 — YELLOW FLAG (낮지만 catastrophic tail)
- *"정신건강 도구가 아님"* 디스클레이머만으로는 부족할 수 있다. 잠재의식 데이터의 유출 / 미성년자 사용 / 자살 신호 오감지 등은 brand에 영구 손상을 줄 수 있는 tail risk.
- GDPR/CCPA + 한국 개인정보보호법은 day 1부터 substantial compliance investment 필요. Pre-seed 예산에서 *"법무"* 항목이 너무 작게 잡혀 있음.

## 4. Diligence Questions (Founder에게 직접 묻는 것)

1. **Retention 가설**: *"Self profile + 친구 5명 케미"* 후 사용자가 90일 retain 하는 hook이 무엇인가? Co-Star의 daily horoscope에 대응하는 *"daily reason to return"* 이 무엇인가?
2. **첫 Creator**: 이미 컨택 시작했는가? 응답률은? *"내 잠재의식을 fan에게 판다"* 는 결정을 받아낸 sample이 있는가?
3. **Pair 검증 결과**: 5–10쌍 실측 시 false positive/negative가 20% 이내로 나오는 baseline이 있는가? 없으면 어떻게 검증할 계획인가?
4. **LLM Unit Economics**: 사용자 1명당 lifetime LLM 비용 실측이 얼마인가? Cache hit ratio target은?
5. **CTO 영입**: 누구를 candidate로 보고 있는가? Mission-driven 채용 메시지에 응답한 후보가 있는가?
6. **Big Tech 진입 시점**: Meta가 같은 카테고리에 진입한다면 founder의 response가 무엇인가? *"속도"* 외에 구체적 plan이 무엇인가?
7. **B2B 매출 (Year 4–5)**: $100M B2B ARR 가정의 buyer는 누구이며, 이미 informal validation conversation을 한 곳이 있는가?
8. **Exit 시나리오**: IPO vs Acquisition. Strategic acquirer로 누구를 상정하는가? (Match Group? Spotify? Apple?)

## 5. Comparable Analysis

| Comparable | Signal과의 유사점 | 결과 |
|---|---|---|
| **Co-Star** (점성술) | Self-discovery, narrative depth, viral GTM | $30M raised, 30M DL, $5M ARR (낮은 monetization) |
| **Replika** (AI companion) | LLM 기반 정서, 깊은 user attachment | $30M+ raised, 10M users, regulatory drama |
| **Character.AI** | LLM 정서 카테고리 | $1B raised, $5B val, monetization unproven |
| **Hinge** | 관계 매칭 + 진정성 brand | $400M acquisition (Match) |
| **Calm / Headspace** | Anti-광고 mission, subscription | $2B / $3B val |
| **MBTI / 16Personalities** | 성격 진단 카테고리 | $20M ARR, 정적 |

**핵심 시사점**: Self-discovery 카테고리에서 viral은 가능하지만 **monetization은 어렵다**. Co-Star가 30M DL에서 $5M ARR에 머물렀다는 것은 Signal의 ARR target이 *"viral은 쉽고 paid conversion이 어려운"* 카테고리 본질에 부딪힐 가능성을 시사. 그러나 Signal의 *"호환성"* 차원은 Co-Star가 갖지 못한 것이고, 이게 paid trigger를 만들 수 있다면 valuation이 다른 곡선을 그린다.

## 6. Investment Recommendation

### Decision Tree

```
Pre-seed $50–100k → Phase 2 검증 (4–6주) →
   ├─ ✅ Pair 검증 통과 + Creator 응답 + LLM cost OK →
   │     → Seed $1–2M LEAD (15–20% target ownership)
   │     → Series A optionality 확보
   └─ ❌ 지표 미달 →
         → Pre-seed graceful exit, learning 보고서
```

### Final Recommendation

**1단계**: **Pre-seed $100k 진입** at $4–6M post.
- Risk: $100k는 잃을 수 있는 사이즈
- Upside: Phase 2 검증 마일스톤 통과 시 Seed lead의 우선권

**2단계**: Phase 2 종료 시 **5개 정량 게이트** 통과 여부 확인:
1. Pair 검증: 5쌍 중 3쌍 이상 *"우리 관계 맞다"* 4+/5
2. Creator 응답률: 컨택 20명 중 4명 이상 응답
3. False positive/negative: 둘 다 < 25%
4. 사용자당 LLM cost projection: < $0.50 / lifetime
5. CTO candidate: shortlist 3명 이상 + 1명 verbal commit

**3단계**: 5개 중 4개 통과 → **Seed $1.5M lead** at $10–12M post (15% ownership)

### Key Risks Underwritten
- Self-discovery 카테고리의 monetization 한계 (mitigated by chemistry trigger)
- Creator cold start (mitigated by founder의 직접 컨택 능력)
- LLM cost curve (mitigated by Layer 2 무비용 + cache)
- Founder dependency (mitigated by seed funding의 첫 사용처가 핵심팀 영입)

### Conviction Level: **6.5 / 10**
*"이건 trillion-dollar 카테고리이거나 $50M ARR ceiling 카테고리거나 둘 중 하나다. 둘 사이에 중간이 거의 없다. Phase 2 검증이 그 결정을 내려준다."*
